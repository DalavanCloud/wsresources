sil-vai mappings

Three TECkit mapping files (compiled and uncompiled) are included 
in this package. 

- SILVai is for use in converting text encoded with 
the SIL Vai font to Unicode. 

- SILVaiE is for use in converting text encoded with the 
SIL Vai Extras font to Unicode. 

- SILVaiNTsfm is intended for use where text has been input
“phonetically” as a syllable (ba, dle, etc.) and conversion 
to Unicode Vai characters is desired (U+A552 VAI SYLLABLE BA, 
U+A514 VAI SYLLABLE DEE, etc.). 

If you wish to be able to display the custom encoding, the 
custom fonts for SILVai and SIL VaiE can be found here:
http://scripts.sil.org/SILVai
