JGLepcha TECkit mapping file
Sept 2011
author: Lorna Priest Evans
email: lorna_evans@sil.org

This mapping file is intended for converting textual data using the JG Lepcha font to Unicode.

This mapping file follows the Unicode storage order for text: C(.)(R)(Y)(V)(F)(^)

Legend:
C = Consonant
. = Nukta (U+1C37)
R = Ra (U+1C25)
Y = Ya (U+1C24)
V = Vowel (U+1C26..U+1C2C)
F = Final Consonant (U+1C2D..U+1C35)
^ = Ran (U+1C36)
